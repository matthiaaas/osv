# osv

Toy operating system and RISC-V emulator written in [V](https://vlang.io) and Rust, respectively.

## Prerequisites

- [`RISC-V GNU Compiler Toolchain`](https://github.com/riscv-collab/riscv-gnu-toolchain/tree/master)
- [V](https://vlang.io)
- Rust

## Building

```
cd kernel
make
```

## Running

```
cd riscv
cargo run
```
